# Stock Trading Platform

## Overview
This is a Java-based Stock Trading Platform developed using Object-Oriented Programming (OOP) concepts.

## Features
- View Market Data
- Buy Stocks
- Sell Stocks
- Portfolio Tracking
- Transaction History

## Technologies Used
- Java
- OOP Concepts
- HashMap
- ArrayList

## Classes Used
- Stock
- Transaction
- User
- StockTradingPlatform

## How to Run

Compile:
javac StockTradingPlatform.java

Run:
java StockTradingPlatform
